# Krátké shrnutí

BrowserRouter – zapíná routování v aplikaci, všechno v něm může používat funkce React Routeru

Routes – vybírá podle URL, která trasa se má vykreslit

Route – deklarují URL cesty a rozhodují, co přesně se má vykreslit

Link – mění URL bez refreshe stránky

useNavigate – navigace z kódu (po akci / logice)

Navigate – přesměrování při renderu komponenty
